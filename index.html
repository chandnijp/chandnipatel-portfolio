Roll Away

A JavaScript game based on who can get to the target quickest.


Basic Rules

  1. 2 player game with 1 dice.
  2. Each player rolls the dice as many times as they wish and then clicks 'hold' to save their score which gets added to their total score.
  3. If the player rolls a 1, their total score goes back to 0.
  4. The first player to reach a score of 100 wins the game.

Advanced Rules

  1. 2 player game with 2 dice.
  2. Each player rolls the dice as many times as they wish and then clicks 'hold' to save their score which gets added to their total score.
  3. If the player rolls a 1, their total score goes back to 0.
  4. By default, the first player to get a total score of 100 wins the game however a new target score can be added in the input field.

  • Date Created: May 2020
  • Front-End: HTML / CSS / JavaScript ES5
  • Visit Github (Basic Version)
  • Visit Github (Advanced Version)
  • Visit Website (Advanced Version)
